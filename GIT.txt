Индекс - специальная область, в которой хранятся изменения файлов, готовые к коммиту
Клонирование репозитория с сервера
git clone path
cd D://project - перейти в проект
ls -la просмотреть содержимое текущей папки
pwd проверить текущую папку
git config --list
git status проверка наличия git репозитория
git init инициализирует git репозиторий
git add . сохранение изменения в GIT
git commit -m 'initial commit' команда фиксирующая изменения в репо
git log просмотр лога
git show отображает информацию о коммите или любом другом объекте git
